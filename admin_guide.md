## 1. Owner Access Gate & Credentials

The Admin Dashboard is now secured behind an **Owner Access Gate**:
- **Authentication Guard**: Visiting `admin.html` without owner credentials displays an authentication screen blocking data access.
- **Login Tabs**: The user icon on the homepage header opens a modal with three tabs: **User Login**, **Admin Login**, and **Sign Up**. Admin login validates credentials and redirects the owner to the dashboard.
- **Admin-Only Elements**: Link references to "Admin Dashboard" in the header menu and footer remain hidden and only display once the Admin/Owner is logged in.
- **Owner Credentials (in `js/config.js`)**:

## 2. Pushing Orders to WhatsApp & Exporting

In the **Checkout Orders** table inside `admin.html`, you have two premium dispatch utilities:
- **Push to WhatsApp**: Click the green **Push** button next to any order. It compiles client details, shipping address, order items list, subtotals, and GST calculations into a clean message, and triggers a direct WhatsApp chat prefilled with this notification!
- **Print PDF Invoice**: Click the white **PDF** button. This opens a premium, print-ready, clean invoice receipt in a print preview panel, allowing you to instantly print physical copies or save them as PDF documents!
- **Export Orders to CSV**: Click the **Export Orders (CSV)** button in the header of the orders table. This downloads all store transactions into an Excel-friendly CSV sheet for bulk bookkeeping!

## 4. Automatic Discount Calculations

When injecting new products on `admin.html`:
- Enter the **Selling Price (₹)** and the **MRP Retail Price (₹)**.
- The system **automatically calculates the discount percentage** (e.g. `40% OFF`) based on these two inputs!
- This discount tag renders instantly on the storefront shop cards and cart sheets without you having to manually input the discount label text.

---

## 5. Product Catalog Visibility & Search

You can search and hide standard or custom products dynamically inside the **Catalog & Inventory** panel on `admin.html`:
- **Search by Product Name**: Type in the search input box to instantly filter the list of products by name.
- **Toggle Store Visibility**: Click the **Eye icon** next to any product (standard or custom). This toggles the product's storefront visibility. When disabled, the product is hidden from the homepage shop grid!
- **Delete Custom Items**: Custom products added through the dashboard will also display a **Trash icon** to delete them permanently from your store database.

## 6. SQL Database Syncing (Supabase PostgreSQL)

For bulk orders and enterprise scalability, you can integrate your shop with a serverless PostgreSQL SQL Database on Supabase:
1. Log into your [Supabase Console](https://supabase.com).
2. Create a new project.
3. Open the **SQL Editor** tab inside Supabase, copy-paste the SQL script below, and click **Run** to generate the order schema:

```sql
CREATE TABLE orders (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  order_id text UNIQUE NOT NULL,
  order_date text NOT NULL,
  customer_email text NOT NULL,
  customer_name text NOT NULL,
  customer_phone text NOT NULL,
  shipping_address jsonb NOT NULL,
  items_ordered jsonb NOT NULL,
  subtotal numeric NOT NULL,
  tax numeric NOT NULL,
  shipping numeric NOT NULL,
  total_amount numeric NOT NULL,
  payment_method text NOT NULL,
  payment_id text NOT NULL,
  payment_status text NOT NULL,
  status text NOT NULL,
  created_at timestamp with time zone DEFAULT timezone('utc'::text, now()) NOT NULL
);
```

4. Go to **Project Settings** -> **API** to copy your **Project URL** and **anon public key**.
5. Paste these credentials into the **SQL Database Connection** form inside `admin.html`, check the **Enable SQL Database Syncing** box, and click **Save Settings**.
6. When enabled, your dashboard indicator will turn green, and all future checkouts will automatically register inside your Supabase PostgreSQL SQL database! The admin panel will also fetch orders directly from this PostgreSQL database.
